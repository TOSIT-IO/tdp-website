Qu'est-ce que TDP?

Trunk Data Platform est une distribution Hadoop, Open Source, totalement libre de droits.

La distribution est co-construite par EDF & La DGFIP (Direction Générale des Finances Publiques), via l'association TOSIT.

TDP est construite à partir des sources Apache des différents composants.



Et pourquoi Trunk?
En référence à la trompe de l'éléphant, logo d'Hadoop.

Contexte

Qu'est-ce qu'Hadoop ?
Hadoop est un écosystème permettant de faire de l'analytique, de la BI*, voir du temps réel,
sur des gros volumes de données.

Qu'est-ce qu'une distribution Hadoop ?
Une distribution est un ensemble cohérent de versions de projets Open Source, fonctionnant parfaitement entre eux.

Que faisons-nous dans TDP ?
Nous partons des sources Apache, nous alignons les versions, compilons les sources,
jouons les tests d'intégrations, générons les binaires et déployons les composants.

Qui utilise Hadoop ?
Hadoop est très utilisé par les grands comptes et les administrations françaises, depuis une dizaine d'années.
Aujourd'hui, Hadoop est principalement utilisé par ceux qui n'ont pas souhaité basculer dans le Cloud Public.

TOSIT

TOSIT est une association française, Loi 1901.

C'est une structure de soutien à l'Open Source, qui vise à soutenir l'émergence de codes,
logiciels et solutions informatiques libres.

Avantages

Avec TDP, vous maîtrisez le cycle de vie de vos infrastructures et de vos applications.

De plus, vous avez la possibilité de définir l'orientation stratégique, en choisissant les projets Apache à intégrer.

TDP est entièrement sécurisé par défaut (SSL/TLS, Kerberos, Ranger, Knox).

Participation

Il y a 5 façons de participer à TDP.

Le plus simple, pour démarrer, est de télécharger notre sandbox et de déployer TDP localement.
Pour cela, il vous faudra un PC sous Linux ou un MacOS avec au minimum 32GB de RAM.
Les compétences nécessaires sont la maîtrise de Linux, de Vagrant, de Virtualbox et d'Hadoop. Github

La seconde façon de participer à TDP est d'être utilisateur final.
Pour cela, nous vous recommandons de vous procurer les services d'un intégrateur, qui se chargera de vous mettre à disposition TDP.

La troisième façon de participer à TDP est de l'installer vous même en Datacenter ou dans le Cloud Public, et d'en être l'administrateur.

La quatrième façon de participer est de devenir contributeur sur le projet.
Nous accueillons tous les bras!

La cinquième et dernière façon de participer est d'en parler autour de vous.
