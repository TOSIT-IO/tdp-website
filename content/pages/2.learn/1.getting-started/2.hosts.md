## `tdp-collection-prerequisites`

Une collection `tdp-collection-prerequisites`, testée avec Rocky 8 et AlmaLinux 8, facilite l'installation les prérequis nécéssaires à TDP sur les hôtes.

Pour un cluster de production, il est recommandé de ne pas utiliser cette collection et d'avoir un cluster PostgreSQL, LDAP et Kerberos hautement disponible.

La commande suivante permet d'installer les prérequis sur l'ensemble des hosts définis dans l'inventaire :

```bash
ansible-playbook ansible_dependencies/ansible_collections/tosit/tdp_prerequisites/playbooks/all.yml
```

Note : Un avertissement concernant les `tdp_vars` peut s'afficher. La collection `tdp-collection-prerequisites` n'utilise pas les `tdp_vars`, cet avertissement peut donc être ignoré.
